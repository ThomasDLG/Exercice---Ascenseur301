LES 3 ELEMENTS INDISPENSABLES POUR BOIRE ME MATE

Le nom de Maté désigne en fait le récipient dans le lequel il est préapré, en forme de calbasse

Remplir dans un récipient "pot à moitié" aux 3/4 inserré la pipette "bambilla"

Faire chauffer l'eau sans la faire bouillir ( entre 70 et 80 ) pour quelle préserve les arômes des l'infusion

En attendant que l'eau soit à bonne température, imprégner légèremnt le maté d'eau froide ou tiède pendant quelques minutes

Verser l'eau chaude autour de la pipette pour l'eau mouille progressivement le maté
